**EliteNest** is a full-stack rental and property management platform designed to simplify the process of finding, listing, and managing rental properties and PG accommodations.

It provides a seamless experience for **tenants, property owners, and administrators** through an intuitive interface, smart filtering, and real-time management features.

---

## 🌟 Project Overview

EliteNest focuses on solving real-world rental challenges by offering:

- 🏡 Easy property discovery based on location and preferences  
- 📅 Hassle-free visit booking and scheduling  
- 💳 Secure payment and transaction tracking  
- 🔧 Efficient maintenance request handling  
- 📊 Dedicated dashboards for different user roles  

---

## 🚀 Features

### 🏠 Property Management

- Advanced search with filters (location, budget, BHK, property type)  
- Interactive **map-based property discovery**  
- Support for apartments, houses, PGs, and co-living spaces  
- Detailed listings with images, amenities, and verification  

---

### 👥 User Roles

- **Tenants**  
  - Browse properties  
  - Book visits  
  - Pay rent  
  - Submit maintenance requests  

- **Property Owners**  
  - List and manage properties  
  - Track bookings  
  - Handle maintenance  

- **Administrators**  
  - Monitor platform activity  
  - Manage users and listings  

---

### 📅 Booking & Scheduling

- Real-time visit scheduling  
- Appointment booking system  
- Digital rental agreements  

---

### 💳 Payments & Transactions

- Online rent payment system  
- Booking fee handling  
- Transaction history tracking  

---

### 🔧 Maintenance Management

- Submit maintenance requests  
- Track request status  
- Owner dashboard for issue resolution  

---

### 📊 Dashboards

- **Tenant Dashboard**: Saved properties, rent details, maintenance  
- **Owner Dashboard**: Listings, bookings, revenue tracking  
- **Admin Dashboard**: Platform analytics and control  

---

### 🎯 Advanced Features

- Location-based recommendations  
- Bachelor-friendly filtering (family, boys, girls, shared)  
- Furnishing options (fully, semi, unfurnished)  
- Amenities filtering (parking, balcony, etc.)  
- Verified property listings
